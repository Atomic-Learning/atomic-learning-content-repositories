Structure of the Repository

The basic structure of the repository is determined by the content template used when creating the repository. Its major components are:

  • content.html: The main HTML file containing the content for the page.
  • metadata.json: The JSON file containing metadata specific to this content page.
  • resources/: A directory which should be used to contain any additional resources, such as images, used in the content. This initially contains an empty file named placeholder.txt which exists to ensure the directory is included in version control and can be ignored.
  • license.md: A file specifying the license of the document. Should not normally be modified.
